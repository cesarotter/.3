Introducción a JavaScript

JavaScript es un lenguaje de programación interpretado, dialecto del estándar ECMAScript.

Se define como orientado a objetos, basado en prototipos, imperativo, débilmente tipado y dinámico.

Algunas características clave de JavaScript incluyen:

  • Lenguaje del lado del cliente
  • Interpretado (no necesita compilación)
  • Multiplataforma
console.log("Hola Mundo");

JavaScript se ejecuta en el navegador web y puede manipular el DOM para crear páginas interactivas.

Variables y constantes

En JavaScript existen tres formas de declarar variables:

  • var - Alcance de función
  • let - Alcance de bloque
  • const - Alcance de bloque y valor constante

Ejemplos de declaración:

var nombre = "Juan"; let edad = 30; const PI = 3.1416;

Las variables declaradas con const no pueden ser reasignadas, aunque sus propiedades pueden modificarse si es un objeto.

Estructuras de control

JavaScript incluye las estructuras de control típicas de los lenguajes de programación:

if (condición) { /* código */ } else { /* código */ }

Estructuras de repetición:

for (let i = 0; i < 10; i++) { /* código */ } while (condición) { /* código */ }

También existe el switch para múltiples condiciones:

switch(expresión) { case valor1: // código break; default: // código }

Las estructuras de control permiten controlar el flujo de ejecución del programa.

Funciones

Las funciones son bloques de código reutilizables que realizan una tarea específica.

Pueden declararse de varias formas:

function saludar() { return "Hola"; } const saludar = function() { return "Hola"; } const saludar = () => "Hola"; // Arrow function

Las funciones pueden recibir parámetros y retornar valores:

function suma(a, b) { return a + b; }

En ES6 se añadieron parámetros por defecto:

function multiplicar(a, b = 1) { return a * b; }
Objetos y arrays

Los objetos son colecciones de propiedades clave-valor:

const persona = { nombre: "Ana", edad: 25, saludar: function() { console.log("Hola"); } };

Los arrays son listas ordenadas de valores:

const colores = ["rojo", "verde", "azul"];

Métodos comunes de arrays:

  • push() - Añade elemento al final
  • pop() - Elimina último elemento
  • map() - Transforma cada elemento

Los objetos y arrays son estructuras fundamentales en JavaScript.
